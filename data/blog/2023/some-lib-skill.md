## axios

axios 是一个轻量的 HTTP 客户端

基于 XMLHttpRequest 服务来执行 HTTP 请求，支持丰富的配置，支持 Promise，支持浏览器端和 Node.js 端。

- 在浏览器端使用 XHR
- 在 node 端使用 http，https
